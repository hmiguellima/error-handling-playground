# neverthrow-experiment

A TypeScript project setup with SWC, ts-node, ts-lint, and neverthrow.

## Setup

```bash
npm install
```

## Scripts

- `npm run dev` - Run the project with ts-node
- `npm run build` - Build with SWC
- `npm run lint` - Lint with tslint
- `npm run clean` - Clean build artifacts

## Dependencies

- **neverthrow** - Safe functional programming with Result type
- **SWC** - Super fast compiler
- **ts-node** - TypeScript execution engine
- **eslint** - Linting framework
